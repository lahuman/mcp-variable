mcp-variable

공공데이터 공통표준용어 사전을 기반으로 한국어 업무 용어와 물리명, 코드 변수명을 변환하는 로컬/HTTP SSE MCP 서버입니다.

엔드포인트

GET /이 가이드 페이지
GET /index.html프로젝트 설명과 연결 가이드
GET /health서버 상태 확인
GET /sseMCP SSE 수신 스트림
POST /messagesMCP JSON-RPC 메시지 전송

MCP 클라이언트 설정

URL 기반 SSE MCP 연결을 지원하는 클라이언트에서는 다음처럼 등록합니다.

{
  "mcpServers": {
    "mcp-variable-sse": {
      "type": "sse",
      "url": "https://term2var.kro.kr/sse",
      "headers": {
        "Authorization": "Bearer <api-key>"
      }
    }
  }
}

API 키 사용

API 키가 설정된 서버에서는 아래 헤더 중 하나를 보내야 합니다.

Authorization: Bearer <api-key>
X-API-Key: <api-key>
실제 키는 서버의 MCP_VARIABLE_API_KEYS 환경변수로만 설정하고, 공개 페이지나 URL query string에 노출하지 마세요.

제공 도구

  • convert_terms: 한국어 업무 용어를 물리명/변수명으로, 또는 물리명을 한국어 용어로 변환합니다.
  • search_terms: CSV 사전에 등록된 용어 행을 검색합니다.
  • suggest_terms: Chroma가 설정된 경우 비슷한 용어, 단어, 도메인을 벡터 기반으로 추천합니다.
입력: 등록일자
출력: regYmd
confidence: exact

Chroma 유사 추천

Docker Compose 배포는 SSE 서버와 Chroma 서버를 함께 실행하고, 서버 시작 시 CSV 캐시 생성과 Chroma 컬렉션 동기화를 처리합니다.

docker compose up -d --build
docker compose logs -f mcp-variable-sse

기본 Chroma 연결 설정은 다음 환경변수로 조정할 수 있습니다.

MCP_VARIABLE_CHROMA_HOST=chroma
MCP_VARIABLE_CHROMA_PORT=8000
MCP_VARIABLE_CHROMA_COLLECTION=mcp_variable_terms
MCP_VARIABLE_CHROMA_SYNC_ON_START=true
Chroma 추천은 후보 탐색용입니다. 표준 변수명 확정은 계속 convert_terms의 사전 기반 exact 또는 composed 결과를 기준으로 판단합니다.

직접 점검

curl https://term2var.kro.kr/health
curl -N -H "Authorization: Bearer <api-key>" https://term2var.kro.kr/sse
