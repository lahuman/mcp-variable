Business naming puzzle

업무 변수명은 네 겹으로 결정됩니다

한글 업무 용어 하나를 코드에 넣기까지 여러 기준이 동시에 맞아야 합니다.

  • 표준 한글명등록일인가, 등록일자인가?
  • 데이터 도메인일자, 일시, 여부, 번호, 코드, 값 중 무엇인가?
  • 물리명 약어REG_DTM, REG_DT 중 무엇이 맞는가?
  • 코드 표기regDtm처럼 팀 코드 스타일에 맞는가?

Dictionary 1

용어 사전

용어 사전은 업무에서 쓰는 표준 한글명을 정합니다.

“등록일”, “작성일”, “생성일”을 각자 쓰는 대신, 표준 용어를 하나로 맞춥니다.
표준 한글명 업무 의미 리뷰 기준

Dictionary 2

도메인 사전

도메인 사전은 값의 의미와 데이터 타입 기준을 정합니다.

일자

YMD, 날짜 8자리처럼 저장 규칙을 함께 떠올립니다.

여부

YN, Y/N 값처럼 의미와 표현 범위를 정합니다.

번호/코드/값

데이터의 역할을 이름 끝에서 읽을 수 있게 합니다.

MCP basics

MCP란?

MCP(Model Context Protocol)는 AI 애플리케이션이 외부 도구와 데이터를 표준 방식으로 연결해 쓰게 해주는 프로토콜입니다.

AI를 더 똑똑하게 만드는 비결은 “더 많이 외우게 하기”가 아니라 “필요할 때 정확한 도구를 호출하게 하기”입니다.

MCP participants

Host, Client, Server

처음에는 세 단어만 기억하면 됩니다.

Host

ChatGPT, Claude, IDE처럼 사용자가 만나는 AI 앱입니다.

Client

Host 안에서 각 MCP Server와 연결을 유지합니다.

Server

도구, 리소스, 프롬프트 같은 실제 기능을 제공합니다.

Skill makes MCP practical

Skill 로 MCP 최대한 활용 하기

MCP가 AI에게 도구를 준다면, Skill은 그 도구를 언제 어떻게 쓸지 알려줍니다.

Skill 없이

AI가 일반 번역, 기억, 취향으로 이름을 만들 수 있습니다.

Skill 있게

skills/mcp-variable-naming/SKILL.md가 사전 조회 순서를 고정합니다.

Skill workflow

Skill이 하는 일

사용자는 목적과 용어를 말하고, Skill은 반복 규칙을 대신 챙깁니다.

용어 수집

코드나 DB 이름이 될 한글 업무 용어를 모읍니다.

결과 판단

Setup step 1

설정 방법

프로젝트를 설치하고 빌드한 뒤, 로컬 서버가 실행되는지 확인합니다.

명령어

npm install
npm run build
node dist/server.js --csv ./data/terms.csv

중요

소스 변경 후에는 다시 빌드합니다. MCP 클라이언트는 dist/server.js를 실행합니다.

Setup step 2

MCP 설정 JSON

AI 클라이언트에 실행 명령과 CSV 경로를 등록합니다.

{
  "mcpServers": {
    "mcp-variable": {
      "command": "node",
      "args": [
        "/Users/k/DEV/mcp-variable/dist/server.js",
        "--csv",
        "/Users/k/DEV/mcp-variable/data/terms.csv"
      ],
      "cwd": "/Users/k/DEV/mcp-variable",
      "disabled": false
    }
  }
}

Usage step 1

사용 방법: DTO 필드명 만들기

코드 식별자는 기본적으로 lowerCamel로 받습니다.

입력

{
  "text": "등록일자\n라우팅결과값",
  "direction": "term_to_physical",
  "outputCase": "lowerCamel"
}

결과 예시

regDtm
rotngRsltVal

Usage step 2

사용 방법: DB 컬럼명 만들기

DB 컬럼, SQL alias, 물리명은 snake로 받습니다.

입력

{
  "text": "등록일자\n라우팅결과값",
  "direction": "term_to_physical",
  "outputCase": "snake"
}

결과 예시
